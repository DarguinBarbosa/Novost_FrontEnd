    Acerca de nosotros

    Novost es un software diseñado para el seguimiento y control de las novedades generadas por los aprendices en el área de teleinformática del Servicio Nacional de Aprendizaje

    Características

    El software esta diseñado con una combinación de tecnologías que benefician tanto al cliente (en el sentido de la respuesta del sistema a sus peticiones) como al programador al tener algunas de las herramientas mas fáciles y utilizadas

    Herramientas

    Novost utiliza una serie de herramientas que son muy famosas actualmente, como lo son: Angular en su parte gráfica o Django en su parte lógica

    Módulos

    El software cuenta con tres módulos:

    01

    Usuarios

    Cuenta con el manejo de todos los usuarios en el sistema, como lo son Aprendiz, Apoyo administrativo, administrador, etc. En esta parte del software se realiza todo el manejo de la información del usuario, validación de datos, control de información, permisos de usuario, entre otros.

    02

    Novedad

    Aquí es donde se realiza la parte importante de nuestro software, se le permite a cada usuario en el sistema generar o procesar una novedad. Aquí se pueden tramitar las novedades de aplazamiento, retiro voluntario, reintegración, cambio de ficha, etc.

    03

    Reportes

    Lo que se realiza en esta parte del software son nada mas que los reportes y gráficas basado en los datos almacenados en el sistema, a su vez, también documentos (PDF, excel, word, etc.) que el software genera por cada novedad que este en proceso.

    Funcionalidades

    Cargar listas

    Esta función permite que mediante listas de excel se puedan cargar los aprendices al sistema.

    Generar una novedad

    Esta función permite que se almacene todo lo relacionado a una novedad y hacerle su respectivo seguimiento.

    Reportar una novedad - Funcionarios

    Esto permite que los funcionarios reporten a algún aprendiz (decersión, condicionamiento, cancelación).

    Revisar solicitudes

    Esta función permite que el apoyo administrativo pueda revisar, procesar y responder una solicitud de novedad.

    Crear novedad - Aprendices

    Esta función permite que los aprendices soliciten una novedad (aplazamiento, retiro voluntario, reintegración).

    Obtener reportes

    Esta función permite que se puedan obtener los reportes parametrizados concernientes a cada novedad y detalle.
